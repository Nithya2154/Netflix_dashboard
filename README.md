## ğŸš€ Getting Started

### Prerequisites
- Microsoft Power BI Desktop (latest version recommended)
- Windows 10 or later / macOS with Power BI Desktop

### Installation & Usage
1. Download the `Netflix_dashboard.pbix` file
2. Open with Power BI Desktop
3. Interact with visuals by:
   - Clicking on chart elements to filter
   - Using slicers to drill down
   - Hovering for detailed tooltips
   - Cross-filtering across visualizations

### Data Refresh
To update the dashboard with new data:
1. Open the `.pbix` file in Power BI Desktop
2. Navigate to Home > Transform Data > Refresh
3. Ensure your data source connection is configured
4. Save and republish if using Power BI Service
